Data types-
There are two types of data types --- 1. Primitive and 2.Non primitve 
The classification is done basedon how Js handles data storage, mutability and referencing. 

Primitive- simple, immutable and are stored by value.
Non primitive- complex, mutable and stored by reference.


PRIMITIVE DATA TYPE
1. string- " "
2. number- 2 to power 53
3. big int- very big numbers
4. boolean- true false
5. null- standalone value 
6. undefined- value not defined yet
7. symbol- unique
